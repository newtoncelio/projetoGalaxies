## 💻 Projeto  
Esse é um projeto Web de uma página que demonstra curiosidades sobre Galáxias.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o curso da Rocketseat com as seguintes tecnologias:

- HTML
- CSS (Modal)
- JavaScript (módulos, eventos)
